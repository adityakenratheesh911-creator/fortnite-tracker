# fortnite-tracker

Fortnite Tracker Discord Bot — AMORISメンバーのPRとイベント結果を毎日Discordに通知します。

## 使い方

| コマンド | 説明 |
|---|---|
| `!stats` | 設定済みプレイヤー全員のステータスを表示 |
| `!player <名前>` | 任意のプレイヤーを検索 |
| `!help_ft` | コマンド一覧を表示 |

## セットアップ

1. `DISCORD_TOKEN` を環境変数に設定
2. `config.py` でプレイヤー名とチャンネルIDを設定
3. `python artifacts/fortnite-bot/bot.py` で起動
